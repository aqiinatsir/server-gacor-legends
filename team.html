Pelajari peran masing-masing staff

Owner

Bertanggung jawab penuh atas operasional server, pengambilan keputusan strategis, dan pengembangan server ke depannya.

🤝

Partner

Membantu owner dalam mengelola server dan berkontribusi dalam pengambilan keputusan penting.

Mengelola storyline, event roleplay, dan memastikan kualitas roleplay di server tetap terjaga.
